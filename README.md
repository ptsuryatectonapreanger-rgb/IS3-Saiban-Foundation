# IS3 Saiban Foundation - E-Commerce Platform

Website e-commerce seperti Shopee dengan fitur lengkap untuk penjualan online.

## Fitur Utama
- 🛍️ Katalog produk dengan pencarian dan filter
- 🛒 Keranjang belanja
- 💳 Checkout dan pembayaran
- 👤 Manajemen profil pengguna
- ⭐ Sistem rating dan review
- 📦 Tracking pesanan
- 💬 Chat dengan penjual

## Tech Stack
- Frontend: React/Next.js + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT

## Cara Memulai
```bash
npm install
npm run dev
```
